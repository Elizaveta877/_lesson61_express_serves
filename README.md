Цей проект реалізує сервер на Node.js та Express.js з використанням архітектури MVC та мідлварів для безпеки та логування.

## Інструкція з установки

1. Клонуйте репозиторій.
2. Встановіть залежності:
   ```bash
   npm install
   ```
   Запустіть сервер:

Bash
node app.js
Опис маршрутів
GET / — Головна сторінка (логування запитів).

GET /users — Отримання списку користувачів (потрібна аутентифікація).

POST /users — Створення користувача (валідація даних + аутентифікація).

GET /articles/:articleId — Отримання статті (перевірка прав доступу).

Використані мідлвари
logRequests: Логування кожного запиту в консоль.

basicAuth: Перевірка заголовка Authorization.

validateUserInput: Перевірка наявності обов'язкових полів.

checkArticleAccess: Контроль доступу до ресурсів статей.
